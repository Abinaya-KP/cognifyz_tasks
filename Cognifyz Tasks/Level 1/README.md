# Cognifyz Technologies Web Development Internship

## Level 1

This repository contains the solutions for **Level 1** of the Cognifyz Technologies Web Development Internship. The projects are developed using **HTML, CSS, and JavaScript** to demonstrate fundamental web development concepts.

---

## Task 1 – HTML & CSS

### Objective

Create a simple webpage using HTML and CSS that includes:

* A heading
* A paragraph
* An image
* A basic form with a single input field and a submit button
* A navigation menu with three links

### Features

* Responsive webpage layout
* Navigation menu
* Web Development Community theme
* Community registration form
* Submit button with a success message
* Clean and modern user interface

### Technologies Used

* HTML5
* CSS3
* JavaScript (for the success message)

---

## Task 2 – JavaScript Basics

### Objective

Build a webpage demonstrating basic JavaScript concepts by implementing:

* A button that changes color when clicked
* A greeting alert based on the current time
* A simple calculator that adds two numbers

### Features

* Interactive button color changer
* Time-based greeting alert
* Simple addition calculator
* User-friendly interface
* Responsive design

### Technologies Used

* HTML5
* CSS3
* JavaScript
